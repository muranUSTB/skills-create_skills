# Skill Creator

专业级 Claude Code Skill 创建工具，帮助用户从零开始设计和实现自定义的 Skill。

## 简介

Skill Creator 是一个功能强大的 Skill 创建助手，它提供：

- **多模板支持**：内置 8 种常见 Skill 模板，覆盖文档处理、数据分析、浏览器操作等多个领域
- **引导式创建**：分步骤引导用户确认所有关键信息，确保创建的 Skill 结构完整、可用
- **完整输出**：自动生成 Skill 目录结构、SKILL.md、README.md、LICENSE.txt 等全部必要文件
- **最佳实践**：遵循 Claude Code Skill 设计规范，生成的 Skill 可直接投入使用

## 功能特性

### 1. 多模板库

提供 8 种预定义模板，满足大多数常见需求：

| 模板名称 | 适用场景 | 复杂度 |
|---------|---------|--------|
| 文档处理类 | PDF、Word、Excel、PPT 等文档的处理操作 | 中等 |
| 数据分析类 | 数据统计、可视化、报表生成等数据处理场景 | 中等 |
| 浏览器操作类 | 网页导航、表单填写、截图采集等 Web 自动化场景 | 较高 |
| 内容创作类 | 写作辅助、翻译、文档生成等创意内容场景 | 简单 |
| 外部服务集成类 | API 调用、第三方服务对接等集成场景 | 较高 |
| 通用工具类 | 文件操作、系统命令、工具调用等通用场景 | 简单 |
| 代码开发类 | 代码生成、重构、审查、调试等开发场景 | 中等 |
| 自定义模板 | 特殊需求、未涵盖场景的完全自定义结构 | 可变 |

### 2. 引导式工作流

采用分步骤引导方式，确保不遗漏任何关键信息：

1. **欢迎与确认**：收集基础信息（名称、描述、作者）
2. **选择模板**：根据需求推荐最适合的模板类型
3. **功能确认**：深入了解功能范围和复杂度
4. **流程定义**：与用户共同定义工作流程
5. **结构确认**：确认输出文件结构
6. **生成文件**：自动生成完整的 Skill 文件
7. **展示确认**：向用户展示并确认生成结果
8. **迭代优化**：根据反馈进行修改
9. **完成交付**：提供使用指导

### 3. 完整文档生成

自动生成以下文件：

- **SKILL.md**：主 Skill 文件，包含完整的功能说明和工作流
- **README.md**：项目说明文档，提供快速入门指南
- **LICENSE.txt**：采用 CC BY-NC-SA 4.0 许可证
- **references/templates/**：模板参考资料目录（根据需要）

## 快速开始

### 使用方式

在 Claude Code 中直接调用 Skill Creator：

```
请帮我创建一个新的 Skill。
```

然后按照引导完成以下步骤：

1. 提供 Skill 名称和描述
2. 选择合适的模板类型
3. 确认功能需求
4. 定义工作流程
5. 确认输出结构
6. 获取完整的 Skill 文件

### 示例对话

```
用户：请帮我创建一个处理 PDF 文件的 Skill。

Skill Creator：好的！让我帮您创建一个 PDF 处理 Skill。

首先，请确认以下信息：
- Skill 名称：pdf-processor
- 描述：高效的 PDF 处理工具，支持读取、转换和合并
- 作者：您的名字

接下来，请选择模板类型：
[展示模板列表]

用户：选择文档处理类模板

Skill Creator：请详细描述您的需求...
[继续引导流程]
```

## 模板详情

### 文档处理类模板

适用于 PDF、Word、Excel、PPT 等文档的处理操作。

**主要功能：**
- 文档读取（文本提取、表格识别、元数据读取）
- 文档转换（格式转换、批量处理）
- 文档生成（模板填充、内容生成）
- 文档分析（关键词提取、摘要生成）

### 数据分析类模板

适用于数据统计、可视化、报表生成等场景。

**主要功能：**
- 数据导入（CSV、Excel、JSON、数据库）
- 数据清洗（缺失值处理、异常值检测）
- 数据统计（描述性统计、分组聚合、相关性分析）
- 数据可视化（折线图、柱状图、散点图等）
- 报表生成（统计报表、分析报告）

### 浏览器操作类模板

适用于网页自动化、测试、截图、数据采集等场景。

**主要功能：**
- 页面导航（打开页面、跳转、多标签管理）
- 元素交互（点击、输入、选择、上传）
- 表单处理（填写、验证、提交）
- 内容提取（文本、表格、图片）
- 截图与录制（页面截图、区域截图）

### 内容创作类模板

适用于写作辅助、翻译、文档生成等场景。

**主要功能：**
- 文档撰写（技术文档、商业报告、文章）
- 内容翻译（多语言翻译、术语管理）
- 内容优化（语法检查、风格优化）
- 模板支持（邮件模板、报告模板）
- 创意生成（标题生成、摘要生成）

### 外部服务集成类模板

适用于 API 调用、第三方服务对接等场景。

**主要功能：**
- 服务连接（连接池管理、连接验证）
- API 调用（GET、POST、RESTful、GraphQL）
- 认证授权（API Key、OAuth、JWT）
- 错误处理（超时、限流、服务不可用）
- 响应处理（数据解析、格式转换）

### 通用工具类模板

适用于文件操作、系统命令调用等通用场景。

**主要功能：**
- 文件操作（读取、写入、复制、移动、删除）
- 系统命令（命令执行、进程管理）
- 网络操作（HTTP 请求、下载、检测）
- 数据处理（格式转换、编码、压缩）

### 代码开发类模板

适用于代码生成、重构、审查等开发场景。

**主要功能：**
- 代码生成（根据需求生成代码、测试用例）
- 代码审查（质量分析、安全检测、优化建议）
- 代码重构（简化代码、模式应用）
- 代码调试（错误分析、日志分析）
- 文档生成（API 文档、注释生成）

### 自定义模板

适用于特殊需求或未涵盖场景的完全自定义结构。

**特点：**
- 灵活的模块组合
- 自定义工作流程
- 自由的功能定义

## 最佳实践

### 文件命名规范

- 使用小写字母和连字符（kebab-case）
- 避免使用空格和特殊字符
- 保持简洁且具有描述性
- 示例：`document-processor`、`data-analyzer`

### 描述编写规范

- **一句话描述**：简洁明了，不超过 20 个字
- **详细描述**：说明功能、用途、特点，长度适中

### 工作流设计原则

- 步骤清晰，每个步骤有明确的目标
- 考虑边界情况和错误处理
- 保持合理的粒度，不过于复杂也不过于简单
- 遵循单一职责原则

### 代码质量标准

- 使用清晰的注释
- 遵循 SOLID 原则
- 保持代码简洁可读
- 包含适当的错误处理

## 常见问题

### Q1：创建的 Skill 可以直接使用吗？

是的，按照本工具生成的 Skill 结构完整，符合 Claude Code Skill 规范，可以直接放入 Skills 目录中使用。

### Q2：可以修改生成的模板吗？

当然可以。在生成文件后，您可以要求任何修改，我们会根据您的反馈进行调整。

### Q3：如何添加自定义模板？

您可以在 `references/templates/` 目录下添加新的模板文件，遵循现有模板的格式规范。

### Q4：生成的 Skill 如何分发？

您可以将整个 Skill 目录打包分享，其他用户只需将其放入自己的 Skills 目录即可使用。

### Q5：支持批量创建多个相关 Skill 吗？

是的，您可以依次创建多个 Skill，或者描述您的批量需求，我们会提供相应的解决方案。
