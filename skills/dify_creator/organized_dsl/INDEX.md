## 📌 推荐学习路径

### 🟢 入门级别（适合新手）

**位置：** `14_参考示例/basic基础模板/`

推荐文件：
- `Agent工具调用.yml` - 基础Agent模式
- `Form表单聊天Demo.yml` - 表单处理
- `json-repair.yml` - JSON处理
- `jieba.yml` - 中文分词
- `53-dify案例分享-12306mcp火车票信息查询-chatflow.yml` - Chatflow基础

### 🟡 中等级别（有一定基础）

**位置：** `01_内容生成与创作/中等_多节点协作/`、`07_Agent与工具调用/`

推荐文件：
- `标题党创作.yml` - 多节点协作
- `翻译_workflow.yml` - 翻译工作流
- `搜索大师.yml` - 搜索Agent
- `MCP.yml` - MCP集成基础

### 🔴 高级级别（复杂工作流）

**位置：** `14_参考示例/complex复杂案例/`

推荐文件：
- `Deep Researcher On Dify .yml` - 深度研究Agent
- `记忆测试.yml` - 复杂记忆系统
- `FLUX绘画机器人+多模态识别+语音播放.yml` - 多模态整合

## 📝 使用说明

1. **查找参考**：根据你想实现的功能，在对应分类目录下查找相似的工作流
2. **学习节点**：查看复杂工作流的节点连接方式，理解流程设计
3. **复用模板**：可直接导入Dify作为基础模板进行修改
4. **组合创新**：多个简单工作流可以组合成复杂应用
