# 通用工具类 Skill 模板

适用于文件操作、系统命令调用、网络请求等通用工具场景。

## 模板结构

```markdown
---
name: [skill-name]
description: [一句话描述]
license: LICENSE-CC-BY-NC-SA 4.0 in LICENSE.txt
author: [作者名]
---

# Skill Name

## 简介

[详细介绍工具功能]

## 功能分类

### 1. 文件操作

文件管理功能：
- 文件读取
- 文件写入
- 文件复制
- 文件移动
- 文件删除
- 目录操作

### 2. 系统命令

系统命令执行：
- 命令执行
- 进程管理
- 环境变量
- 系统信息

### 3. 网络操作

网络相关功能：
- HTTP 请求
- 下载文件
- 网络检测
- 端口检测

### 4. 数据处理

数据处理功能：
- 格式转换
- 数据编码
- 数据压缩
- 校验计算

## 工作流

### Step 1：解析请求

解析用户指令：
- 识别操作类型
- 提取参数
- 验证权限

### Step 2：执行操作

执行具体操作：
- 调用系统工具
- 执行文件操作
- 发送网络请求

### Step 3：处理结果

处理返回结果：
- 格式化输出
- 错误处理
- 结果验证

## 使用示例

```markdown
请读取「config.json」文件内容。

请执行「npm install」命令。

请下载「https://example.com/file.pdf」。

请将「data.csv」转换为 JSON 格式。
```

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| operation | string | 是 | 操作类型 |
| target | string | 是 | 目标路径/URL |
| options | object | 否 | 附加选项 |

## 输出格式

| 输出类型 | 格式 | 说明 |
|---------|------|------|
| 内容 | string | 文件/命令输出 |
| 文件 | 文件 | 生成的文件 |
| 状态 | object | 操作状态 |

## 注意事项

- 文件操作需要权限验证
- 系统命令需要安全过滤
- 网络请求需要超时控制
- 敏感操作需要确认
- 危险命令需要警告

## 安全建议

- 白名单限制可用命令
- 路径遍历防护
- 输入参数过滤
- 操作日志记录
```
