# 浏览器操作类 Skill 模板

适用于网页自动化、测试、截图、数据采集等浏览器操作场景。

## 模板结构

```markdown
---
name: [skill-name]
description: [一句话描述]
license: LICENSE-CC-BY-NC-SA 4.0 in LICENSE.txt
author: [作者名]
---

# Skill Name

## 简介

[详细介绍浏览器操作的功能]

## 功能特性

### 1. 页面导航

浏览器导航功能：
- 打开指定 URL
- 页面跳转
- 返回/前进
- 多标签页管理

### 2. 元素交互

页面元素操作：
- 点击操作
- 文本输入
- 下拉选择
- 文件上传
- 滚动操作

### 3. 表单处理

表单相关操作：
- 表单填写
- 表单验证
- 表单提交
- 多步骤表单

### 4. 内容提取

页面内容获取：
- 文本提取
- 表格提取
- 图片下载
- 链接采集

### 5. 截图与录制

视觉输出功能：
- 页面截图
- 区域截图
- 全屏截图
- 视频录制

### 6. 等待与条件

等待机制：
- 显式等待
- 隐式等待
- 条件等待
- 页面加载等待

## 工作流

### Step 1：初始化浏览器

启动浏览器实例：
- 配置浏览器选项
- 设置窗口大小
- 加载用户配置

### Step 2：导航到目标

访问目标页面：
- 输入 URL
- 处理重定向
- 等待页面加载

### Step 3：执行操作

执行用户指定的操作：
- 元素定位
- 交互操作
- 数据提取

### Step 4：处理结果

处理和返回结果：
- 截图保存
- 数据提取
- 状态报告

### Step 5：清理资源

关闭浏览器：
- 清理会话
- 关闭浏览器
- 释放资源

## 使用示例

```markdown
请打开「https://example.com」并截图保存。

请填写表单字段：
- 姓名：张三
- 邮箱：zhangsan@example.com

请采集「https://news.example.com」的所有文章标题。
```

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| url | string | 是 | 目标页面 URL |
| operation | string | 是 | 操作类型 |
| target | string | 否 | 目标元素选择器 |
| options | object | 否 | 附加选项 |

## 输出格式

| 输出类型 | 格式 | 说明 |
|---------|------|------|
| 截图 | PNG | 页面截图 |
| 文本 | string | 页面内容 |
| 数据 | JSON | 提取的结构化数据 |
| 文件 | 文件 | 下载的文件 |

## 注意事项

- 尊重网站的 robots.txt 协议
- 控制请求频率，避免封禁
- 登录态需要妥善管理
- 动态内容需要合理等待

## 依赖工具

- Playwright
- Puppeteer
- Selenium
```

## 安全注意事项

- 避免在浏览器中暴露敏感信息
- 使用完毕后及时关闭浏览器
- 合理设置超时时间
- 处理证书和安全性问题
```
