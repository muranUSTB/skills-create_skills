# 数据分析类 Skill 模板

适用于数据统计、可视化、报表生成等数据处理场景。

## 模板结构

```markdown
---
name: [skill-name]
description: [一句话描述]
license: LICENSE-CC-BY-NC-SA 4.0 in LICENSE.txt
author: [作者名]
---

# Skill Name

## 简介

[详细介绍数据分析的功能]

## 功能特性

### 1. 数据导入

支持多种数据源：
- CSV 文件
- Excel 文件
- JSON 数据
- 数据库连接
- API 数据获取

### 2. 数据清洗

数据预处理功能：
- 缺失值处理
- 异常值检测
- 数据类型转换
- 重复数据去除

### 3. 数据统计

统计分析功能：
- 描述性统计
- 分组聚合
- 相关性分析
- 趋势分析

### 4. 数据可视化

图表生成功能：
- 折线图
- 柱状图
- 散点图
- 饼图
- 热力图

### 5. 报表生成

报告输出功能：
- 统计报表
- 分析报告
- 数据导出

## 工作流

### Step 1：数据获取

从指定来源获取数据：
- 读取本地文件
- 调用 API
- 数据库查询

### Step 2：数据处理

清洗和预处理数据：
- 数据验证
- 格式标准化
- 异常处理

### Step 3：数据分析

执行分析操作：
- 统计分析
- 趋势预测
- 模式识别

### Step 4：结果输出

生成分析结果：
- 可视化图表
- 数据报告
- 结构化数据

## 使用示例

```markdown
请分析「sales_data.csv」中的销售趋势。

请对「user_behavior.json」进行用户分群分析。

请生成一份月度数据报表，包含关键指标和趋势图。
```

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| data_source | string | 是 | 数据来源路径或连接信息 |
| analysis_type | string | 是 | 分析类型 |
| options | object | 否 | 分析选项 |

## 输出格式

| 输出类型 | 格式 | 说明 |
|---------|------|------|
| 图表 | PNG/SVG | 可视化结果 |
| 数据 | CSV/JSON | 分析结果数据 |
| 报告 | Markdown/PDF | 结构化报告 |

## 注意事项

- 大数据集可能需要分批处理
- 敏感数据需要进行脱敏处理
- 统计分析结果需注明置信度

## 依赖工具

- Pandas（数据处理）
- NumPy（数值计算）
- Matplotlib/Plotly（可视化）
- SciPy（统计分析）
```
