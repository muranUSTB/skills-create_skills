# 内容创作类 Skill 模板

适用于写作辅助、翻译、文档生成、创意内容创作等场景。

## 模板结构

```markdown
---
name: [skill-name]
description: [一句话描述]
license: LICENSE-CC-BY-NC-SA 4.0 in LICENSE.txt
author: [作者名]
---

# Skill Name

## 简介

[详细介绍内容创作的功能]

## 功能特性

### 1. 文档撰写

各类文档创作：
- 技术文档
- 商业报告
- 新闻文章
- 博客内容
- 营销文案

### 2. 内容翻译

多语言翻译功能：
- 文本翻译
- 文档翻译
- 术语管理
- 风格保持

### 3. 内容优化

文本优化功能：
- 语法检查
- 风格优化
- 可读性提升
- SEO 优化

### 4. 模板支持

预设模板功能：
- 邮件模板
- 报告模板
- 社交媒体模板
- 商务文档模板

### 5. 创意生成

创意内容生成：
- 标题生成
- 摘要生成
- 关键词提取
- 大纲生成

## 工作流

### Step 1：理解需求

分析用户需求：
- 识别写作类型
- 确定目标受众
- 理解风格要求

### Step 2：内容规划

规划内容结构：
- 生成大纲
- 确定关键点
- 选择写作风格

### Step 3：内容生成

执行内容创作：
- 按大纲撰写
- 融入专业知识
- 保持风格一致

### Step 4：内容优化

优化生成的内容：
- 语法检查
- 逻辑检查
- 风格调整

### Step 5：输出交付

输出最终内容：
- 格式化输出
- 提供多种格式
- 添加使用建议

## 使用示例

```markdown
请帮我写一封商务邮件，关于项目进度汇报。

请将这段中文翻译成英文，保持专业语气。

请帮我优化这段文案，使其更有说服力。

请根据以下要点生成一篇博客文章...
```

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| content_type | string | 是 | 内容类型 |
| content | string | 是 | 输入内容 |
| style | string | 否 | 写作风格 |
| target | string | 否 | 目标受众 |
| options | object | 否 | 附加选项 |

## 输出格式

| 输出类型 | 格式 | 说明 |
|---------|------|------|
| 文本 | Markdown | 标准输出格式 |
| 摘要 | string | 概要信息 |
| 大纲 | 列表 | 内容结构 |

## 注意事项

- 保持内容的原创性
- 注意版权和引用规范
- 敏感内容需要谨慎处理
- 专业术语需要准确使用

## 质量标准

- 语法正确性
- 逻辑连贯性
- 风格一致性
- 信息准确性
```
