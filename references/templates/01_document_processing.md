# 文档处理类 Skill 模板

适用于 PDF、Word、Excel、PPT 等文档的处理操作。

## 模板结构

```markdown
---
name: [skill-name]
description: [一句话描述]
license: LICENSE-CC-BY-NC-SA 4.0 in LICENSE.txt
author: [作者名]
---

# Skill Name

## 简介

[详细介绍文档处理的功能]

## 支持的格式

- PDF
- Word（.docx）
- Excel（.xlsx）
- PPT（.pptx）
- [其他格式]

## 主要功能

### 1. 文档读取

读取各种格式的文档内容，支持：
- 文本提取
- 表格识别
- 图片提取
- 元数据读取

### 2. 文档转换

文档格式转换功能：
- PDF 转 Word
- Word 转 PDF
- Excel 转 CSV
- [其他转换]

### 3. 文档生成

创建新文档：
- 模板填充
- 内容生成
- 格式设置

### 4. 文档分析

文档内容分析：
- 关键词提取
- 摘要生成
- 结构分析

## 工作流

### Step 1：接收输入

接收用户提供的文档和操作指令：
- 验证文件格式
- 检查文件可读性
- 解析用户指令

### Step 2：执行处理

根据指令执行相应的处理操作：
- 文档读取
- 格式转换
- 内容处理
- 错误处理

### Step 3：输出结果

返回处理结果：
- 文本内容
- 生成的文件
- 处理状态报告

## 使用示例

```markdown
请帮我读取「document.pdf」并提取所有文本内容。

请将「report.docx」转换为 PDF 格式。

请根据模板生成一份合同文档，公司名称为「ABC公司」。
```

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| file_path | string | 是 | 文档路径 |
| operation | string | 是 | 操作类型 |
| options | object | 否 | 附加选项 |

## 输出格式

根据操作类型返回相应结果：
- 文本内容：直接返回文本
- 文件转换：返回文件路径
- 分析报告：返回结构化数据

## 注意事项

- 限制文件大小（建议不超过 50MB）
- 支持加密文件的读取（需提供密码）
- 大文件处理可能需要较长时间

## 依赖工具

- Python-docx（Word 处理）
- PyPDF2（PDF 处理）
- OpenPyXL（Excel 处理）
- python-pptx（PPT 处理）
```

## 模板变量说明

| 变量 | 说明 | 示例 |
|-----|------|------|
| skill-name | Skill 名称 | document-processor |
| 一句话描述 | 功能简述 | 高效的文档处理工具，支持多种格式的读取、转换和分析 |
| 作者名 | 开发者署名 | 沐然 |
